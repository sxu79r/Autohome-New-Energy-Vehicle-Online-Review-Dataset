# Autohome-New-Energy-Vehicle-Online-Review-Dataset
汽车之家新能源汽车在线评论数据集，其中有7w条做了情感标注,4w条未标注。标注规则:选取高星评论为积极评论，低星评论为消极评论。评论爬取时间:2024.7.11
